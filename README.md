# Namaste React
React application. This is a playground project for POCs of new react libraries/frameworks.

# Parcel (Refer parceljs.org)
- Dev build
- Local Server
- HMR = Hot Module Replacement
- File watching algorithm - Written in C++
- Caching for faster builds
- Image optimisation
- Minification
- Bundling
- Compressing
- Differential bundling - bundle to support different versions of browser
- Diagnostics 
- Error handling
- HTTPs
- Tree shaking = Remove unused code/files
- Different dev (`npx parcel index.html`) and prod builds (`npx parcel build index.html`)
